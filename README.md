# Mechanized Weed Remover

A battery-powered agricultural machine developed to reduce manual labor in weed removal operations. The system utilizes a motor-driven hydraulic mechanism to uproot weeds efficiently while improving productivity and supporting sustainable farming practices.

## Key Features

* 12V rechargeable battery-powered operation
* Hydraulic weed extraction mechanism
* Solar charging support
* Lightweight and portable design
* Suitable for various agricultural fields
